# Weather App

A weather application that provides accurate current weather conditions, hourly forecasts, and a 7-day forecast for locations around the world. Users can search for any city, switch between measurement units, and view detailed weather information.

## Features

- Search weather by city or location
- Current weather conditions
- Current, feels-like, minimum, and maximum temperatures
- Humidity, wind speed, and precipitation information
- Hourly weather forecast
- 7-day weather forecast
- Select different days to view hourly forecasts
- Toggle between Metric and Imperial units
- Fully responsive design
- Keyboard accessible with hover and focus states

## Built With

### Frontend

- HTML5
- CSS3
- JavaScript
- React
- TypeScript

### APIs

- Open-Meteo Weather API
- Open-Meteo Geocoding API

## Getting Started

Clone the repository:

```bash
git clone https://github.com/nghuthjo/weather-app.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
